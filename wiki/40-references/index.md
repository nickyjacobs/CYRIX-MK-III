# References

Externe documentatie, ingestueerd via `/ingest`. Anders dan projecten/knowledge zijn references niet-persoonlijk (publieke docs), dus committable.

## Collecties

- [[40-references/claude-code/overview|Claude Code]] — officiële Claude Code documentatie (slash-commands, settings, overview, ...)
- `dutchquill/` — DutchQuill schrijfstijl-gidsen
- [[40-references/log|Ingest-log]] — append-only audit-trail van wat wanneer is ingestueerd

## Conventie

Reference-pagina's gebruiken `ingest-date` en `review-date` (geen `created`/`updated`), plus `source-url`. Zie `_templates/reference-page.md`. De `wiki-librarian` monthly-audit flagt references waarvan de `review-date` verstreken is.
